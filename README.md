# 📝 Blog Generator

A Blog Generator application that helps users generate well-structured blog posts using AI. The application allows users to enter a topic, choose preferences, and instantly receive a complete blog article.

---

## 🚀 Features

- Generate blogs on any topic
- Clean and responsive user interface
- AI-powered content generation
- Fast response time
- Easy-to-use design
- Copy generated content
- Download generated blogs (if available)

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/blog-generator.git
```

### 2. Navigate into the project

```bash
cd blog-generator
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add your API key

Create a `.env` file and add:

```env
API_KEY=your_api_key_here
```

### 5. Run the application

```bash
python app.py
```

---

## 💡 How It Works

1. Enter a blog topic.
2. Select the desired options.
3. Click **Generate**.
4. The AI generates a complete blog.
5. Read, copy, or download the generated content.
